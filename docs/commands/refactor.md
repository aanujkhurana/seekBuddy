# /refactor Command

Use this command to improve structure without changing behaviour.

## Rules

- No new features.
- No behaviour changes.
- No unrelated formatting churn.
- Keep commits small.
- Update imports.
- Run checks.

## Good Refactors

- Move selectors into one file.
- Extract AI provider interface.
- Split large Electron IPC file.
- Extract config loader.
- Extract logger.
