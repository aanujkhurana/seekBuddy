# /start Command

Use this command when beginning work from a simple script repository.

## Process

1. Read `START.md`.
2. Read all key docs.
3. Inspect repository.
4. Create first issue for Electron desktop shell.
5. Create branch `feature/electron-desktop-shell`.
6. Implement shell only.
7. Do not add AI yet unless desktop shell is done.
8. Prepare PR.

## First Feature

Electron desktop shell with:
- config form
- resume picker
- start script
- stop script
- logs
